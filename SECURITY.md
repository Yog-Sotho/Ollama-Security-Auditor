# 🔒 Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.4.x   | ✅ Yes    |
| 1.3.x   | ⚠️ Yes    |
| < 1.3   | ❌ No     |

## Reporting a Vulnerability

I take the security of **Ollama Security Auditor** seriously. Since this tool is designed for security research, we encourage responsible disclosure.

- **Report via:** [GitHub Security Advisories](https://github.com/Yog-Sotho/Ollama-Security-Auditor/security/advisories/new) or contact us at **[Insert Your Email Here]**.
- **What to include:** 
  - Description of the vulnerability.
  - Steps to reproduce.
  - Impact assessment.
  - Suggested fix (if any).

I aim to acknowledge reports within 48 hours and resolve critical issues within 7 days.

## Usage Disclaimer
This tool is intended for authorized security testing only. The maintainers are not responsible for misuse of this software.
